# RDFSharp.Stores
<b><a href="https://github.com/mdesalvo/RDFSharp">RDFSharp</a></b> extensions for storage of RDF data. Actual implementation provides support for: 
<ul>
    <li>SQLite</li>
    <li>Firebird</li>
    <li>MySQL</li>
    <li>PostgreSQL</li>
    <li>Oracle</li>
</ul>

</ul>
