# RDFSharp.RDFStore
<b><a href="https://github.com/mdesalvo/RDFSharp">RDFSharp</a></b> extensions to back RDF data on common SQL relational databases. Actual implementation provides support for: 
<ul>
    <li>SQLite</li>
    <li>Firebird</li>
    <li>MySQL</li>
    <li>PostgreSQL</li>
</ul>
